# Design system

FundedFence is dark-first, calm, and risk-focused. It uses deep green-black surfaces, restrained mint for healthy states, amber for caution, and red only for breach/danger. Panels use subtle borders rather than heavy shadows; typography carries the hierarchy.

## Tokens

- Canvas: `#07100d`; panel: `#0d1a15`; raised panel: `#111f19`.
- Primary text: `#eef3ef`; muted text: `#8f9d96`.
- Healthy/accent: `#9ae6bd` / `#63d69c`.
- Caution: `#e8c57a`; danger: `#ef7d7d`.
- Radius: 9–18 px by control hierarchy.
- Body: Geist/Segoe UI fallback; large product headings use compact negative tracking; occasional Georgia italic is limited to brand emphasis.

## Components

Button hierarchy is primary mint, secondary bordered, and quiet text. Status pills always include text, not color alone. The risk ring includes a numeric score and label. Connection state combines dot, text, and freshness. Data tables collapse to a mobile summary without hiding core market, position, P&L, or direction.

## Accessibility

Semantic landmarks and labels are required. Focus rings use high-contrast mint. Inputs retain native keyboard behavior. The layout adapts at 1100, 820, and 560 px. Reduced-motion preference removes meaningful transitions. Illustrative data and unverified rules are announced as text, not only badges.
